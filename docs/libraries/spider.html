Spider类

这个类的主要作用是根据提供的URL，来抓取这个URL的内容，返回的是一个数组。

初始化

Spider类在控制器里完成初始化工作，函数：$this->load->library,不过在初始化的时候，需要提供一个array数组，这个数组包含一些基本的设置：user-agent，url，header，cookie，POST 或者GET方法，timeout，http_header，http_referer,post_field,file

$this->load->library('spider', $spider_config);

$spider_config是一个array数组,例如：$spider_config = array('url' => 'http://ip.taobao.com/service/getIpInfo.php?ip='，'http_header'=>'')

函数参考

$this->spider->__construct()

这个是这个spider类的构造函数，主要作用是判断是否有CURL扩展，以及初始化类

$this->spider->initialize()

初始化spider类，根据提供的array数组，构建成spider需要的变量名格式。因为spider类内部的变量是私有的，所以构建出来的变量名开头都有“_”这个符号

$this->spider->exec()

这个函数根据提供的URL，一直array数组的相关设置，抓取并返回一个数组

$this->spider->post()

将数据提交的方法设置成POST方法

$this->spider->get()

将数据提交的方法设置成GET方法

$this->spider->set_cookies()

如果有需要话，设置cookies

$this->spider->set_user_agent()

如果有需要话，重新设置user-agent

$this->spider->set_method()

设置提交方法，默认是GET

$this->spider->set_url()

设置要抓取的URL

$this->spider->set_file()

设置文件的路径
