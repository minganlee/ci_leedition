IP 辅助函数

IP辅助函数文件包含了获得IP地址相关信息的函数

加载该辅助函数

本辅助函数通过如下代码加载：

$this->load->helper('ip');

可用函数如下：

ip_location('ip_string')

ip_string是一个包含IP地址的字符串（string），例如“36.250.226.6”

函数使用例子：

ip_location('36.250.226.6');

注意：请将“36.250.226.6” 这个IP地址替换为需要处理的IP地址！

IP辅助函数的将返回一个数组（array）,如下所示：

  int(0)
  ["data"]=>
  array(13) {
    ["country"]=>
    string(6) "中国"

    string(6) "350000"
    ["city"]=>
    string(9) "福州市"

其中code的值的含义为，0：成功，1：失败。

valid_ip('$ip_string')

该函数用于判断IP地址是否是有效的IP地址，如果是有效IP地址则返回TRUE,无效则返回FALSE。
